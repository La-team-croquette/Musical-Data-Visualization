Les Données

Un peu d'information

Source ?

Les données d'écoute sont directement extraites de nos informations personnelles de nos comptes Spotify.

Les cerles ?

Le graphique représente les liens entre les utilisateurs par les musiques qu'ils écoutent. Une grande balle est une personne, et une plus petite une musique.

Les liens ?

Des personnes sont liées par une musique si elles l'ont toutes les deux écoutées.

Depuis quand ?

Les données regroupent un an d'écoute. C'est donc la fenêtre d'affichage maximal.
